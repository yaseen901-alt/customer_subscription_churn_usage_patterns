# Customer Churn Prediction

Predict customer churn from subscription and usage patterns using machine learning.

## Dataset
- CSV file: `customer_subscription_churn_usage_patterns.csv`
- Binary target: `churn` (Yes/No)
- Features include numeric usage metrics and categorical subscription details.
